## Installation & Usage

To reproduce these results, you need a Python environment (like Google Colab or local VS Code).

1. Clone the repository:
   ```bash
   git clone https://github.com/Slava2201/ACT-Theory.git
   ```
2. Install dependencies:
   ```bash
   pip install numpy scipy matplotlib
   ```
3. Run the engine:
   ```bash
   python octant_spectrum.py
   ```
